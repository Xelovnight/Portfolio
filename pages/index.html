Les logiciels que je maîtrise.

Visual Studio Code

Visual Studio Code est un éditeur de code source léger et puissant développé par Microsoft. Il est particulièrement apprécié pour sa flexibilité, son extensibilité et sa prise en charge de nombreux langages de programmation, grâce à une vaste bibliothèque d'extensions. Il offre des fonctionnalités avancées telles que la coloration syntaxique, l'autocomplétion, le débogage intégré, la gestion de versions et la collaboration en temps réel, ce qui en fait un outil populaire pour les développeurs de logiciels.

Docker

Docker est une plateforme de virtualisation légère qui permet de créer, déployer et exécuter des applications dans des environnements isolés appelés conteneurs. Ces conteneurs sont des unités portables qui incluent tout le nécessaire pour faire tourner une application, ce qui garantit qu'une application fonctionne de la même manière, peu importe où elle est exécutée. Docker facilite ainsi la gestion des applications, en permettant leur déploiement rapide et cohérent sur différents systèmes ou serveurs.

Suite Office

La suite Office 365 de Microsoft est un ensemble d'outils de productivité basés sur le cloud, incluant des applications comme Word, Excel, PowerPoint, et Outlook. Elle facilite la création, la collaboration et le partage de documents, de feuilles de calcul et de présentations en temps réel. En outre, elle offre des services supplémentaires tels que OneDrive pour le stockage en ligne et Teams pour la communication et la collaboration.

GitLab

GitLab est une plateforme de gestion du cycle de vie des applications DevOps qui intègre le contrôle de version Git, la gestion des tickets, l'intégration continue et le déploiement continu. Avec une interface conviviale et des fonctionnalités complètes, GitLab offre un environnement collaboratif pour le développement logiciel, de la planification initiale au déploiement final, le tout sur une seule plateforme.

GitHub

GitHub est une plateforme de développement collaboratif qui héberge du code source et gère le contrôle de version en utilisant Git. Elle propose des outils pour l'intégration continue, la gestion de Projets et la collaboration entre développeurs. GitHub facilite le travail d'équipe avec des fonctionnalités telles que les pull requests, les issues, et les discussions.

SolidWorks

SolidWorks est un logiciel de CAO (Conception Assistée par Ordinateur) utilisé pour la modélisation 3D, la simulation et la documentation de produits. Il offre une interface conviviale, des outils puissants pour la conception mécanique, et des fonctionnalités avancées telles que l'analyse de mouvement et la création de dessins détaillés.
